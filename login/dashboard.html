Control de roles y permisos

Desde este panel podrás decidir qué usuario puede ver, crear, editar o eliminar dentro de cada módulo del ERP. La vista está lista como maqueta funcional para conectar después con backend y Azure SQL.

Seguridad central

Super Administrador con control total del sistema.

Este rol está reservado para el creador del ERP y no debe depender de permisos operativos normales. Su función es administrar usuarios, roles, módulos y configuraciones críticas del sistema.

Cómo debería funcionar en backend
  • El rol `Super Administrador` debe omitir todas las restricciones operativas normales.
  • Los permisos base viven por rol y los ajustes finos viven por usuario y módulo.
  • El login debe cargar rol, permisos heredados y sobreescrituras para construir el menú permitido.
Siguiente implementación

  • Guardar cambios del panel en Azure SQL.
  • Aplicar permisos reales al dashboard y a cada módulo.
