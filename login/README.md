# Visualizar el proyecto

## Vista actual

La carpeta `login` ya incluye:

- `index.html`: acceso principal
- `dashboard.html`: dashboard de demostracion

## Como probar rapido

Sin instalar nada, abre `login/index.html`.

Al capturar empresa, usuario y contraseĂąa, la vista navega a `dashboard.html`.

## Con Electron

1. Entra a `login`
2. Ejecuta `npm install`
3. Ejecuta `npm start`
